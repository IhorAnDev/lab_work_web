6.2 Контрольні питання
1.	Для чого використовують доповнення Firebug?
Для дослідження властивостей фрагментів html сторінки з урахуванням стилів.
2.	Наведіть синтаксис правила в CSS?
Селектор {властивість: значення; }
3.	Назвіть способи застосування таблиці стилів до HTML документа?
Підключити файл css, написати блок <style>, в атрибуті тега style="".
4.	Наведіть властивості тексту?
Назва шрифту, розмір, насиченість, стиль шрифту, відступ, вирівнювання, висота рядка, відстань між словами, 
відстань між буквами, регістр, оформлення тексту, тінь, пробіли, колір.
5.	Наведіть властивості шрифту?
Назва, насиченість, розмір, стиль, розтягнуте накреслення.
6.	Наведіть властивості кольору і тла?
color: color|initial|inherit|currentColor|transparent|unset;

    background-color

    background-image

    background-position

    background-size

    background-repeat

    background-origin

    background-clip

    background-attachment
